## Crazyな機能の説明:

昨日以前のToDoが自動的に完了になるようにしています。

意図としては、このToDoアプリ自体をタスク管理としてではなく、今日行うことの宣誓として使うためです。そのため、前日のToDoが完了していることを前提の動作にしています。

## 動作確認方法:

`npm run dev`を実行して、ブラウザで`http://localhost:3000`にアクセスしてください。

CRUD操作を確認する手順は以下の通りです。

- C: タイトルなどを入力後、プラスボタンを押下する
- R: ToDo追加後の、ページ読み込み時
- U: ToDo上の編集ボタンを押下し、編集を行い、保存ボタンを押す
- D: ToDo上のバツボタンを押下する

※編集ボタンなどはホバーすると出てきます。

## 技術選定の理由:

create-next-appから特に変更していません。

## AI活用の範囲:

最初に自分で型とコンポーネント（ToDoBar）の定義をし、その後のUIはAIで生成したものを手動で少し修正しています。（日本語に直したりなど）

その後のAPI定義と呼び出しなどは手動で実装しています。

ToDoの自動完了はエラー内容の要約だけAIに聞いています。
